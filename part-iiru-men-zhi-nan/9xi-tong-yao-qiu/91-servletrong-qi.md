### 9.1 Servlet容器

以下嵌入式servlet容器可以直接使用：  


| 名称 | Servlet版本 | Java版本 |
| --- | --- | --- |
| Tomcat 8  | 3.1 | Java 7+ |
| Tomcat 7  | 3.0 | Java 6+ |
| Jetty 9.3 | 3.1 | Java 8+ |
| Jetty 9.2 | 3.1 | Java 7+ |
| Jetty 8   | 3.0 | Java 6+ |
| Jetty 8   | 3.1 | Java 7+ |

您还可以将Spring Boot应用程序部署到任何兼容Servlet 3.0+ 的容器中。
